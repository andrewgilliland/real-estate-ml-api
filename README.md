# Real Estate ML API

## Libraries Included

### pandas

Load and manipulate datasets (CSV, JSON, etc.)

### numpy

Numeric operations and matrix math

### scikit-learn

Training, preprocessing, and evaluating ML models

### joblib

Save and load trained models efficiently

### fastapi

Build the API (request validation, routing, docs)

### uvicorn

Lightweight ASGI server to run your FastAPI app

### pydantic

Data validation and schema enforcement (comes with FastAPI)

### python-dotenv

Manage environment variables (for config, API keys, etc.)

### ruff

Linting and formatting (fast replacement for flake8 + black)

### pytest

Testing routes and model logic

### httpx

Test your API with HTTP requests
